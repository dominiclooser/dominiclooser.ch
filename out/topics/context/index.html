Commands

A command begins with Backslash (). A command in ConTeX is the same as a macro in TeX. A macro package is a collection of macros/commands.

command = setup([]) + scope({})

\define[1]\mpThree{The ID3 Tag of the MP3 file is: #1}

Setup

\setuppapersize
\setuplayout
\setuppagenumbering
\setuphead[which][textstyle=]
